Detailed Description

RigidBody is an implementation of the DeviceInterface virtual class for OptiTrack rigid bodies. This device assumes that a VRPN client is used to broadcast the rigid bodies as geometry_msgs::PoseStamped over the ROS topic specified by topic_name.
